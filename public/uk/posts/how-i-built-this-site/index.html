Технології#

Цей сайт розміщено з допомогою сервісу github pages та побудовано з використанням інструменту hugo плюс hugo-PaperMod тема.

Hugo#

Я обрав Hugo, тому що він швидкий та простий. Заповнення вмісту сайту в основному відбувається з допомогою файлів з розміткою Markdown. Також в Hugo присутня інтернаціоналізація (можливість вести сайт на двох мовах) “з коробки”, що є великим плюсом, як на мене.

Github pages#

Надає готовий до використання CI/CD процес завдяки, чому мій сайт оновлюється автоматично після кожного коміту до репозиторію.1

Вартість#

Я витратився лише на купівлю домену kurotych.com. Хостинг являється безкоштовним.

Корисні посилання#

  • Hosting Hugo on Github
  • GitHub Pages with Namecheap custom domain

  1. Ви повинні мати налаштований github workflow у вашому репозиторію з сайтом. Дивись gh-pages.yml ↩︎
